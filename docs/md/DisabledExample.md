```html
<CheckPicker
  data={data}
  defaultValue={'Julius'}
  disabled
/>
```

> 添加 `disabled` 属性即可让 `CheckPicker` 处于禁用状态。


```html
<CheckPicker
  data={data}
  defaultValue={'Julius'}
  disabledItemValues={['Eugenia', 'Travon', 'Vincenza']}
/>
```

> 配置 `disabledItemValues` 属性，可以让部分选项处于禁用状态.


>[示例代码](https://github.com/rsuite/rsuite-checkpicker/blob/master/docs/DisabledExample.js)
